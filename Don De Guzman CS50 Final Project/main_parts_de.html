Die Gehirne

Steht für "Central Processing Unit". Die CPU ist die Hauptkomponente eines Computers, der Anweisungen verarbeitet.
Es führt das Betriebssystem und die Anwendungen aus und erhält ständig Eingaben vom Benutzer oder von aktiven Softwareprogrammen.
Es verarbeitet die Daten und erzeugt eine Ausgabe, die von einer Anwendung gespeichert oder auf dem Bildschirm angezeigt werden kann.

Die Nerven

Die Hauptplatine ist die Hauptplatine Ihres Computers und wird auch als Hauptplatine oder Logikplatine bezeichnet.
Wenn Sie jemals Ihren Computer öffnen, ist das Motherboard das größte Stück Silizium, das Sie sehen.
An das Motherboard angeschlossen finden Sie die CPU, das ROM, die Speicher-RAM-Erweiterungssteckplätze, die PCI-Steckplätze und die USB-Anschlüsse.
Es enthält auch Controller für Geräte wie Festplatte, DVD-Laufwerk, Tastatur und Maus.
BGrundsätzlich sorgt das Motherboard dafür, dass alles in Ihrem Computer zusammenarbeitet.

Der Muskel

Der größte Teil der Verarbeitung auf einem Computer erfolgt über die Zentraleinheit oder CPU des Computers.
Um der CPU eine Pause zu geben und einen effizienteren Betrieb zu ermöglichen, kann eine Grafikkarte verwendet werden, um den Grafikteil der Verarbeitungslast zu verarbeiten.
Da die meisten heutigen Programme grafisch ausgerichtet sind, kann die Grafikkarte dazu beitragen, dass fast jedes Programm effizienter ausgeführt wird.
Der Leistungsunterschied macht sich jedoch insbesondere bei Bildbearbeitungsanwendungen und 3D-Spielen bemerkbar.

Das Wissen

Ein Computerspeichergerät ist jede Art von Hardware, die Daten speichert.
Der häufigste Speichergerätetyp, über den fast alle Computer verfügen, ist eine Festplatte.
Auf der primären Festplatte des Computers werden das Betriebssystem, Anwendungen sowie Dateien und Ordner für Benutzer des Computers gespeichert.

Die Energie

Ein Netzteil ist eine Hardwarekomponente, die ein elektrisches Gerät mit Strom versorgt.
Es wird von einer Steckdose mit Strom versorgt und wandelt den Strom von Wechselstrom in Wechselstrom um, was der Computer benötigt.
Außerdem wird die Spannung auf einen angemessenen Wert geregelt, sodass der Computer reibungslos und ohne Überhitzung läuft.
Das Netzteil ist ein wesentlicher Bestandteil jedes Computers und muss ordnungsgemäß funktionieren, damit die restlichen Komponenten funktionieren.
